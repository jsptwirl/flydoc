# 扫码登录功能

### 在配置中心配置参数`supportqrcodelogin`支持扫码登录

```
// 扫码登录支持控制
// true表示支持，false表示不支持, 默认值为false
{
    "supportqrcodelogin": "false"
}
```

### 进入扫码登录页面，进行登录

1. 点击按钮  
![](http://apaas.wxchina.com:8881/wp-content/uploads/%E6%89%AB%E7%A0%81%E7%99%BB%E5%BD%951.png)
2. 打开玄讯app进行扫码登录
