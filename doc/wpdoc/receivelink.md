# receivelink

接收参数的行为，目前该行为只能放置在 `initial` 节点中，且也是目前唯一能放入 `initial` 节点中的行为。

```json
{
    "code": "896903044652994649",
    "type": "receivelink",
    "desc": "参数接收",
    "param": {
        "datatype": "0",
        "name": "__linkparam",
        "ctrlcode": "",
        "properties": [
            {
                "name": "id",
                "alias": "storeid",
                "ctrl": {
                    "component": "",
                    "code": ""
                }
            }
        ]
    }
}
```
