# 新时间表达式

* 支持周单位
* 支持链式写法
* 在uiflycode上使用，加上System命名空间，如
  ```
  System.year(0)
  System.year(-1).thismonth(0, 5)
  // 赋值到日期控件
  let ctrl = Page.getCtrl('日期控件')
  ctrl.value = new Date(System.year(1)).getTime()
  ```
