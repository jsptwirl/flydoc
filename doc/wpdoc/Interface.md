## Interface

# Interface

该区域为放置接口的区域，即该区域放置的事件都不是由当前表单触发的，而是通过接收其他表单发出的广播信息进行触发。

放置在该区域的事件都有一个额外的 `key` 属性：

```json
{
    "presenter": {
        "interface": [
            {
                ......
                "desc": "外部数据变动导致本页面刷新",
                "key": "refreshlist",
                ......
                "actions": [
                    ......
                ]
            }
        ]
    }
}
```

* key

  其内容是一个字符串，用于记录其所接收的广播事件的关键字。一个表单内不能出现重复的关键字，多个表单可以使用同一关键字。

接口事件的其余配置与普通的事件一致。
