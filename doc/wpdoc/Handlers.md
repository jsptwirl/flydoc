## Handlers

表单内部逻辑的放置区域，该区域的事件通常由控件触发，例如点击按钮，列表加载数据等。

handlers 中可以有多个独立的事件，每个事件由一系列的行为构成。

```json
{
    "presenter": {
        "handlers": [
            {
                "code": "546131346436513"
                "desc": "event1",
                "actions": [
                    ......
                ]
            }
        ]
    }
}
```
